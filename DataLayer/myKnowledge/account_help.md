# Account & Login Help

## Creating an Account
Users can sign up using:
- Email and password
- Google account

## Password Reset
- Click "Forgot Password"
- Enter registered email
- Reset link will be sent

## Account Issues
If you cannot log in:
- Check email/password
- Reset password
- Contact support if issue persists

## Profile Updates
Users can update:
- Address
- Phone number
- Preferences
